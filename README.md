# 📘 VocaBuddy

작심삼일을 방지하고 지속적인 영어 학습을 유도하기 위한  
Gamification 기반 영단어 학습 웹 애플리케이션입니다.

---

## 🚀 주요 기능

### 🔐 인증 (Auth)
- 로그인 (이메일 / 소셜 로그인 UI)
- 회원가입 (관리자 / 일반 사용자 구분)
- 비밀번호 찾기

### 📚 단어장 (Voca)
- TOEIC 단어장 제공 (50개 이상)
- 단어장 목록 조회
- 단어 학습 (암기 모드)
- 단어 테스트 (뜻 → 영어 입력)

### 🛠 관리자 기능 (Admin)
- 단어장 생성 / 수정 / 삭제
- 단어 추가 / 수정 / 삭제

### 👤 사용자 (Member)
- 프로필 조회
- 프로필 수정
- 레벨 및 경험치 표시

### 🧩 기타
- 하단 네비게이션 기반 앱 구조
- 상점 / 친구 기능 (UI만 구현, 추후 개발 예정)

---

## 🛠 개발 스택

### Frontend
- React (Vite)
- Tailwind CSS

## 📝 Commit Message Convention

| 타입 | 설명 |
|------|------|
| ✨ feat | 새로운 기능 추가 |
| 🐛 fix | 버그 수정 |
| 📝 docs | 문서 추가, 수정, 삭제 |
| ✅ test | 테스트 코드 추가, 수정, 삭제 |
| 🎨 style | 코드 형식 변경 |
| ♻️ refactor | 코드 리팩토링 |
| ⚡ perf | 성능 개선 |
| 💚 ci | CI 관련 설정 수정 |
| 🔧 chore | 기타 변경사항 |
| 🔥 remove | 코드 및 파일 제거 |

## 🧱 아키텍쳐 구조
src/
├─ app/ # 페이지 및 UI 구성
│ ├─ components/ # 공통 UI 컴포넌트
│ ├─ pages/ # 화면 단위
│ 
├─ main/ # 기능 관련한 파일들 위치
│ ├─features/ # 도메인별 기능 로직
│    ├─ domain/
│     ├─ auth/ # 인증 관련 기능
│     ├─ member/ # 사용자 관련 기능
│     └─ voca/ # 단어장 관련 기능
│
├─ styles/ # 스타일 설정
│
└─ test/ # 테스트용 더미데이터, 추후 삭제
